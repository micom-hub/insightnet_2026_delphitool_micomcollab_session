# InsightNet 2026 MICOM & Delphi Collaboration Tool Session

This repository contains the worksheets and flyer for the collaborative Tool Session between MICOM and Delphi, highlighting Delphi data and tooling capacity. 

This session occurred on Tuesday, April 28th, from 1:00-2:30pm EST.

For more information about the conference https://insightnet.us/am2026/

Files: 

* Delphi Tooling flyer - InsightNet April 2026.pdf: Informational flyer about the data systems and tools made by the Delphi team. Includes QR codes to additional information
* miniproject1.pdf - Worksheet for working through using the Epidata Portal (data system) offered by Delphi
* miniproject2.pdf - Building off of Mini-Project 1 (but still able to be completed independently), a worksheet for working through the analysis and data summary tools offered by Delphi
* miniproject3.pdf - Building off of Mini-Project 2 (but still able to be completed independently), a worksheet for working through the forecasting toodls offered by Delphi
